About This Project

Welcome to the Mythology Bestiary! This project was created as part of the CodePath Web Development curriculum to explore data organization, responsive web design, and dynamic theme switching.

My goal is to provide a clean, searchable interface for enthusiasts to learn about legendary creatures from across global folklore. Each entry includes a brief description, origin, and known weaknesses, along with an image to bring the myths to life.
